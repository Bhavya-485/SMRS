# SMRS – Smart Medical Reminder System

SMRS is an **ESP32-based Smart Medical Reminder System** that helps elderly people to remember to take medicines on time.

## Feature

* Remainders set in **App**
* Medicine reminder using **buzzer and led**
* **OLED display** notifications and wrist band vibrates by **erm coin motor.**
* **BLE connection** between medicine box and wrist band
* Works with **ESP32-C3**

## Hardware

* ESP32C3 DEV KIT
* ESP32-C3 SuperMini
* OLED Display (SSD1306)
* LED
* ERM COIN MOTOR
* Buzzer
* Push Button
* LIPO and LI-ION BATTERIES
